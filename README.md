## Gradle 이란?  
 Gradle은 Groovy를 기반으로 Ant와 Maven과 같은 이전 세대 빌드 도구의 단점을 보완을 한 빌드 도구.   
 의존성이나 플러그인 설정 등과 같은 빌드에 필요한 설정을 하게 된다.
